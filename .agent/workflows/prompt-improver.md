# Prompt Improver

Optimize the provided prompt (`$ARGUMENTS`) following Claude 4 best practices.

1. **Analyze Dimensions**
   - **Explicitness:** Specific vs vague.
   - **Context & Motivation:** Why is this needed?
   - **Action Clarity:** Implement vs research.
   - **Format Control:** Output requirements.
   - **Anti-Hallucination:** Verification steps.
   - **Complexity Calibration:** Scoping.

2. **Apply Transformations**
   - Specific, detailed instructions.
   - Context/Motivation ("because...").
   - Explicit implementation intent.
   - Positive framing.
   - Investigation/verification anchors.
   - Structured approach for complex tasks.

3. **Output**
   - **Analysis:** Key weaknesses.
   - **Improvements Applied:** List of changes.
   - **Optimized Prompt:** Ready-to-use prompt.
   - **Tips:** Advice for this prompt type.
