# Amelia Presentation Image Generation Prompts

> Detailed prompts for creating slide images about the Amelia agentic coding orchestrator architecture.

**Design System:**
- **Color Palette:** Dark theme (#1a1a1a background), sage green (#5B8A72), gold (#FFC857), white/light gray text
- **Aesthetic:** Clean, modern, professional tech visualization
- **Style:** Geometric, data visualization, flowing diagrams
- **Tone:** Corporate-appropriate, sophisticated, not cartoonish

---

## 1. Title Slide - "Amelia: Intelligent Execution Model"

**Title:** Orchestrated Intelligence Abstract

**Prompt:**
Create a sophisticated abstract visualization representing AI-powered code orchestration on a dark background (#1a1a1a). At the center, visualize a glowing neural network node in sage green (#5B8A72) with flowing data streams radiating outward in elegant curves. The data streams should transition from sage green to gold (#FFC857) as they extend outward, representing the flow of information through the system. Include subtle geometric patterns in the background - hexagons and connecting lines forming a faint grid structure. The composition should suggest intelligence, flow, and systematic organization without being literal. Add particle effects along the data streams to suggest active computation. The overall mood should be futuristic yet professional, with a sense of controlled power and precision. Use depth of field to create focus on the central node while the outer streams fade into soft bokeh. Include subtle light rays emanating from the core, creating a sense of enlightenment and insight. The image should evoke themes of automation, intelligence, and systematic orchestration.

**Aspect Ratio:** 16:9

**Key Visual Elements:**
- Central neural network node (sage green)
- Radiating data streams (gradient from sage to gold)
- Hexagonal geometric background pattern
- Particle effects suggesting computation
- Light rays and depth of field effects
- Dark professional background

**Color Notes:** Primary sage green (#5B8A72) at center, transitioning to gold (#FFC857) at edges, on dark background (#1a1a1a)

## Usage Notes

**For AI Image Generators:**
- These prompts are optimized for tools like Midjourney, DALL-E 3, Stable Diffusion XL
- Recommended additions: `--ar 16:9` (aspect ratio), `--style raw` (for photorealistic tools), `--v 6` (for Midjourney)
- For Stable Diffusion: Add quality tags like `highly detailed, professional, 8k, sharp focus, volumetric lighting`

**For Manual Design:**
- Use these as creative briefs for graphic designers
- Reference tools: Figma, Adobe Illustrator, Sketch
- Consider animated versions for interactive presentations

**Consistency:**
- Maintain color palette across all slides: sage green (#5B8A72), gold (#FFC857), dark background (#1a1a1a)
- Use consistent iconography style (outlined vs. filled)
- Keep visual complexity balanced - not too minimal, not too cluttered

**Typography Recommendations:**
- Sans-serif fonts: Inter, SF Pro, or Roboto for labels and descriptions
- Monospace: JetBrains Mono or Fira Code for code snippets
- Ensure sufficient contrast on dark backgrounds (WCAG AA minimum)
